Table keys

description
%exact [CV] unitigs aligning to reference using suffix trees
%align [CV] unitigs aligning to reference using BWA
precision [CV] number of kmers in the unitigs that appear in the reference
recall [CV] number of kmers in the reference that appear in the unitigs
f-score [CV] 2x(recall + precision/(recall * precision)
s-recall [CV] number of kmers appearing in one haplotype only that appear in the unitigs
total kmers [UE] total number of distinct kmers
good [UE] number of kmers both in reference and in unitigs
erroneous [UE] number of kmers in the unitigs not in the reference
missing [UE] number of kmers in the reference not in the unitigs
err_rate [UE] error rate
miss_rate [UE] missing rate
#unitigs [UE] number of unitigs
unmapped [AB] number of unitigs BWA couldn't map
map N50 [AB] N50 of mapped unitigs
#bp [AB] number of breakpoints in mapped unitigs
